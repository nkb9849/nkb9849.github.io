# 2. 내용

Git 환경설정 시 사용했던 명령어 정리

| 명령어                                                    | 설명                             |
| --------------------------------------------------------- | -------------------------------- |
| git init                                                  | 사용하는 git 저장소 초기화       |
| git config --global user.name “닉네임”                    | 닉네임 설정                      |
| git config --global user.email [깃허브 아이디(메일 형식)] | 이메일 설정                      |
| git add [파일명]                                          | 커밋하기 원하는 파일 선택        |
| git commit -m "[메시지 내용]"                             | 메시지를 달아 하나의 버전 만들기 |
| git log                                                   | 생성한 커밋 보기                 |
